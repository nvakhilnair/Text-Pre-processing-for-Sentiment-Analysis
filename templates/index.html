Text-Pre-processing-for-Sentiment-Analysis

    API Endpoint

        https://text-pre-processing.herokuapp.com/

Of all data, the text is the most unstructured form and so means we have a lot of cleaning to do. This API help convert noise from high dimensional features to the low dimensional space to obtain as much accurate information as possible from the text.

For any qery or issues. Please contact me madewithpy009@gmail.com

Usage


# Here is a curl example
curl -X POST https://text-pre-processing.herokuapp.com -H "Content-Type: application/json" -d "{\"text\":\"I need to watch the Entertaiment but power is not there\",\"corrector\":\"True\"}"

# Here is a python example
import requests
from requests.structures import CaseInsensitiveDict
url = "https://text-pre-processing.herokuapp.com"
headers = CaseInsensitiveDict()
headers["Content-Type"] = "application/json"
data = '{"text":"I need to watch the Entertaiment but power is not there","corrector":"True"}'
resp = requests.post(url, headers=headers, data=data)
print(resp.status_code)

To perform preprocessing you need to make a POST call to the below URL. Input data should be in JSON format containing 2 parameter which are explained below both are mandatory fields :
https://text-pre-processing.herokuapp.com/ 



Result example :

{
    "text": "watch entertain power",
    "corrector": "True"
}

QUERY PARAMETERS

Field Type Description
text String Input Text data
corrector Boolean If you need auto spelling correct while text preprocessing set to True else False
